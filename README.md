# 3DGameProto

Proyek ini adalah prototype engine/game menggunakan **C++**, **Vulkan**, dan **SDL3** untuk membuat mekanik dasar serta prototype simulasi **realistic armor**.  
Masih dalam tahap **Work in Progress (WIP)**.

## Menggunakan
- C++20
- Vulkan
- SDL3
- CMake

## Status
Work in Progress belum bisa digunakan/fitur belum selesai.

## Cara Build
Gunakan **CMake**, untuk sekarang G:\ArmorSimulation\build\ArmorSimulation.exe. execute itu.
